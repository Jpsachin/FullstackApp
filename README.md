# FullstackApp

This is a fullstack project using Node.js (latest), React, and Vite (latest).

## Structure

- `client/` — React frontend powered by Vite
- `server/` — Node.js backend

## Setup Instructions

### Prerequisites

- Node.js (latest recommended)

### Setup

1. Install dependencies for both client and server:
   ```sh
   cd client && npm install
   cd ../server && npm install
   ```
2. Start the development servers:
   - Client: `npm run dev` (inside `client`)
   - Server: `npm start` (inside `server`)

## More

- Customize your app in the respective folders.
- For more info, see the documentation for [Vite](https://vitejs.dev/) and [Node.js](https://nodejs.org/).
